## 🔥 Де стежити за концертами

**Світ**

- [**Songkick**](https://www.songkick.com/) — підписка на артистів і сповіщення про шоу поруч.
- [**Eventbrite**](https://www.eventbrite.com/) — афіші по містах; відфільтруйте музичні події.
- [**LiveNation**](https://www.livenation.com/) / [**Ticketmaster**](https://www.ticketmaster.com/) — великі тури й офіційні старти продажу (завжди звіряйте продавця з вашим регіоном).
